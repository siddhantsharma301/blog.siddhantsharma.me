# Blog Converter

Markdown to HTML blog converter with YAML frontmatter support.

## Usage

```bash
python3 convert.py path/to/your/blog-post.md
```

## Features

- YAML frontmatter support for metadata (title, authors, dates)
- Multiple authors with optional URL links
- Automatic styling and template wrapping
- Pandoc-based markdown conversion

## Installation

```bash
uv sync
```
